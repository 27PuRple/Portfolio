### 작품 설명
+ 아두이노와 BLDC 모터를 사용하여 기존 스케이트보드를 전동 스케이트보드로 제작.
+ 애플리케이션을 제작하여 블루투스 통신으로 스케이트보드 조작.
+ 가속도 센서를 사용하여 좌우로 기울어지는 각도를 측정하여 탑승자가 넘어질 것 같으면 속도를 줄이고 Buzzer로 경고음 알림.
