# QR코드를 통한 주문 시스템 & SLAM을 활용한 서빙로봇
## 프로젝트 개요
푸드테크(Food + Technology) + 언택트(Un + Contact) 문화가 발전하면서 무인 시스템이 꾸준히 증가함.
팀 프로젝트 중 서빙로봇 개발 파트를 담당하여 해당 내용을 기재.
